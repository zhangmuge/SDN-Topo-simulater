# SDN-TOPO-SIMILATE

项目灵感来源:[Thomaash/me: SDN topology editor in your web browser with Mininet, image and addressing plan export. (github.com)](https://github.com/Thomaash/me)



## 使用方法:

构建Docker:

```sh
docker-compose
```



运行:

```sh
npm run install
npm run serve
```

根据前端界面构建拓扑图,并点击提交.拓扑图会提交给后端进行模拟拓扑的运行.后端内置了一些简单的脚本以供测试.
